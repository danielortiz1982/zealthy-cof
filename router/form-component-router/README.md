# Form Component Router

> Schema

```javascript
const FormComponentSchema = new Schema({
  name: {
    type: String,
  },
  htmlID: {
    type: String,
  },
  htmlType: {
    type: String,
  },
  label: {
    type: String,
  },
});
```

> Post Data

```json
{
  "name": "about-me",
  "htmlID": "about-me",
  "htmlType": "text",
  "label": "About me"
}
```
